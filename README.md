# 🏦 Bank Management System

A C++ console application designed to simulate a simple banking system using Object-Oriented Programming concepts and data structures.

---

# 🚀 Features

- Create new bank accounts
- Deposit money
- Withdraw money
- Display account details
- Search for accounts
- Simple banking operations

---

# 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Data Structures

---

# 📚 Concepts Implemented

- Classes & Objects
- Encapsulation
- Functions
- Arrays / Structures
- Searching & Sorting
- File Management

---

# ▶️ How to Run

1. Open the project in VS Code or any C++ IDE
2. Compile the code
3. Run the executable

Example:

```bash
g++ main.cpp -o bank
./bank
```
